Netflix EDA and Recommendation System

Recommendation Engine: The engine is expected to carry out 4 fairly simple tasks:

  • Get the index of the requested show title.
  • Compute the cosine similarity of all shows to this show.
  • Sort the results from step 2 from the highest cosine similarity to lowest and get the top 5.
  • Return the shows titles and similarity confidence (cosine similarity) in JSON format.

Flask API:

API and server should work in the following way:

  • Load title column as an index dataframe.
  • Load previously saved TF-I
